Integration Model
→
Report-Only Pattern
Teleradiology providers integrate into the originating hospital's RIS/PACS. Images flow out, reports flow back. Integration complexity sits at the hospital end.
Four distinct public RIS/PACS environments across the country: Agfa (Auckland), Philips Vue RIS/PACS (Northland PACS-only + Te Manawa Taki + Central), COMRAD/Intelerad (Te Waipounamu excl. Southern), Karisma/Sectra (Southern). A national integration standard must accommodate all four, plus the differing integration layers between public (Rhapsody / DICOM direct) and private (KailoHub).
