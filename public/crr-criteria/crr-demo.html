CRR Decision Support Tools

Two prototype tools to support community-referred radiology referrals. Select a tool below to preview it in this pane.

📋 Criteria Viewer

A read-only reference tool that displays the National Radiology Access Criteria for a selected exam and body site, with regional HealthPathways localisation. Designed to sit alongside a referral form so the referrer can check whether their clinical scenario meets funded access criteria.

🔍 Triage Advisor

An AI-assisted tool that evaluates a draft referral note against the access criteria. The referrer pastes or types their clinical presentation and receives feedback on whether the note demonstrates criteria are met, what might be missing, and suggestions to strengthen the referral before submission.

⚠ Concept Demonstration

These are prototypes for evaluation and feedback. HealthPathways remains the authoritative source for radiology access criteria. Criteria shown here are derived from the National Radiology Access Criteria and should not be used for clinical decisions until validated.
